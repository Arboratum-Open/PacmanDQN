# PacmanDQN
Deep Reinforcement Learning in Pac-man

## Usage

To train a network on 10000 games and play 100 games on a medium-sized map, run:
```
python3 pacman.py -p PacmanDQN -n 10100 -x 10000 -l mediumClassic
```

## Requirements

- `python==3.5.1`
- `tensorflow==0.8rc`
